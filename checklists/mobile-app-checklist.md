# Checklist — testowanie aplikacji mobilnej

## Instalacja i start

- [ ] Instalacja czysta
- [ ] Aktualizacja z poprzedniej wersji
- [ ] Pierwsze uruchomienie
- [ ] Ekran splash/loading
- [ ] Obsługa braku internetu
- [ ] Obsługa wolnego internetu

## Konto i sesja

- [ ] Logowanie poprawne
- [ ] Logowanie błędne
- [ ] Wylogowanie
- [ ] Wygasła sesja
- [ ] Reset hasła
- [ ] Blokada konta / limity prób

## Mobile-specific

- [ ] Background/foreground
- [ ] Przerwanie przez połączenie
- [ ] Zmiana orientacji
- [ ] Różne rozdzielczości
- [ ] Tryb ciemny/jasny
- [ ] Uprawnienia: kamera, lokalizacja, powiadomienia, pliki
- [ ] Push notifications
- [ ] Deep links
- [ ] Lokalizacja językowa

## Dane i API

- [ ] Poprawne dane
- [ ] Brak danych
- [ ] Bardzo dużo danych
- [ ] Błędny response
- [ ] Timeout
- [ ] 401/403/404/500
- [ ] Ponowienie requestu

## UX

- [ ] Komunikaty błędów
- [ ] Ładowanie
- [ ] Disabled states
- [ ] Empty states
- [ ] Potwierdzenia akcji destrukcyjnych
