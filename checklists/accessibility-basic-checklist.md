# Checklist — podstawy accessibility mobile

- [ ] Tekst ma wystarczający kontrast
- [ ] Przyciski mają odpowiedni rozmiar dotykowy
- [ ] Elementy interaktywne mają czytelne etykiety
- [ ] Aplikacja działa z TalkBack/VoiceOver na głównych ekranach
- [ ] Formularze mają komunikaty błędów powiązane z polami
- [ ] Nie polegamy tylko na kolorze
- [ ] Dynamic font / większy tekst nie psuje layoutu
- [ ] Kolejność fokusu jest logiczna
