# Prompty do GitHub Copilot / AI w IDE

## Zrozumienie testowanego kodu

```text
Explain this module from QA perspective.
Focus on:
- main business rules,
- input validation,
- edge cases,
- external dependencies,
- what should be tested manually,
- what should be automated.
```

## Generowanie testów Playwright

```text
Create Playwright tests for this user flow.
Use stable locators.
Add clear test names.
Avoid brittle waits.
Add comments only where they explain business intent.
```

## Generowanie testów Appium

```text
Create an Appium test skeleton for this mobile flow.
Use Page Object pattern.
Separate locators from test logic.
Add setup assumptions and required test data.
```

## Review testów

```text
Review this automated test as a senior QA automation engineer.
Find:
- flakiness risks,
- bad waits,
- weak assertions,
- duplicated code,
- missing negative checks,
- improvements in naming and structure.
```
