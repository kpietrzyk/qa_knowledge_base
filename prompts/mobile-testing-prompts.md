# Prompty mobile QA

## Checklist mobile feature

```text
Jesteś seniorem Mobile QA. Przygotuj checklistę testową dla tej funkcji mobilnej.

Uwzględnij:
- Android,
- iOS,
- różne rozdzielczości,
- offline/online,
- słaby internet,
- uprawnienia,
- orientację ekranu,
- powrót z background,
- push notifications,
- deep links,
- edge cases.

Funkcja:
[OPIS]
```

## Analiza logcat

```text
Przeanalizuj logcat z Androida.
Wskaż:
- możliwą przyczynę błędu,
- najważniejsze linie,
- czy to crash, ANR, problem sieciowy czy problem danych,
- co dołączyć do bug reportu.

Log:
[WKLEJ]
```

## Analiza request/response

```text
Jesteś QA z doświadczeniem w API i mobile. Przeanalizuj request/response.
Wskaż:
- czy request wygląda poprawnie,
- czy response pasuje do expected result,
- możliwe błędy backendu,
- możliwe błędy po stronie aplikacji,
- dodatkowe testy.

Request/response:
[WKLEJ]
```
