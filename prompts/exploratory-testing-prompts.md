# Prompty do exploratory testing

## Session based testing

```text
Przygotuj kartę sesji eksploracyjnej dla funkcji:
[FUNKCJA]

Uwzględnij:
- cel sesji,
- zakres,
- ryzyka,
- dane testowe,
- heurystyki,
- notatki do zebrania,
- potencjalne bugi,
- pytania do product ownera.
```

## Heurystyki

```text
Zaproponuj heurystyki testowe dla aplikacji mobilnej:
[OPIS APLIKACJI]

Uwzględnij:
- CRUD,
- permissions,
- interruptions,
- network,
- data consistency,
- device constraints,
- security basics,
- accessibility.
```
