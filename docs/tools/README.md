# Tools

Ten folder zawiera opisy narzędzi. Każdy plik ma podobną strukturę, żeby człowiek i AI-agent mogli szybko porównać narzędzia.

## Kategorie

- `mobile/` — Android, iOS, emulatory, ADB, scrcpy, Appium, Maestro, nagrywanie, inspekcja elementów
- `api/` — Postman, Bruno, Hoppscotch, HTTP Toolkit, Reqable
- `web/` — Playwright i narzędzia webowe
- `ai/` — Copilot, Continue, MCP, agenci, Testsigma, MaestroGPT
- `debugging/` — proxy, crash reporting, raporty
- `test-management/` — Jira, Xray, Kiwi TCMS, TestLink, Qase, Testomat.io, AgileTest
- `bug-reporting/` — Jam, Bird Eats Bug, Greenshot, Loom, Instabug
- `ui-inspection/` — Layout Inspector, Figma Dev Mode, Accessibility Scanner, uiautomatorviewer
