# Sources and verification notes

Repo zawiera wiedzę praktyczną oraz linki do oficjalnych stron/repozytoriów narzędzi.

## Weryfikowane narzędzia

- Appium — open-source automation framework dla mobile/desktop/IoT: https://github.com/appium/appium
- Appium Inspector — GUI do znajdowania selektorów: https://github.com/appium/appium-inspector
- Maestro — open-source framework E2E dla Android/iOS/web, YAML flows: https://github.com/mobile-dev-inc/maestro
- Playwright — framework automatyzacji web z obsługą Chromium/Firefox/WebKit: https://github.com/microsoft/playwright
- Browser Use — framework dla AI-agentów używających przeglądarki: https://github.com/browser-use/browser-use
- Appium MCP — MCP server dla mobile automation i AI-assisted Appium: https://github.com/appium/appium-mcp
- scrcpy — mirror i nagrywanie Androida: https://github.com/Genymobile/scrcpy
- HTTP Toolkit — proxy dla Android i desktop: https://httptoolkit.com/
- Reqable — all-in-one API client + proxy: https://reqable.com/
- Bruno — kolekcje API jako pliki w Git: https://github.com/usebruno/bruno
- Jam.dev — 1-klik bug reporting z logami: https://jam.dev/
- Greenshot — screenshoty z adnotacjami: https://getgreenshot.org/
- Qase — nowoczesny TMS z integracją Jira: https://qase.io/
- Testomat.io — manual + automation TMS: https://testomat.io/
- AgileTest — test management natywnie w Jira: https://marketplace.atlassian.com/apps/1222095/agiletest-test-management-for-jira
- Testsigma — AI-driven test automation: https://testsigma.com/
- Kiwi TCMS — open-source TMS self-hosted: https://github.com/kiwitcms/Kiwi
