# Ścieżka Junior QA → Mid QA

## Junior

- poprawne bug reporty,
- manualne test cases,
- podstawowe API,
- podstawy SQL,
- ADB/logcat,
- praca w Jira.

## Junior+

- rozumienie architektury aplikacji,
- analiza request/response,
- testy regresyjne,
- podstawy CI,
- pierwsze testy Maestro/Playwright.

## Mid

- plan testów,
- ryzyka,
- strategia regresji,
- priorytetyzacja,
- automatyzacja sensownych przypadków,
- mentoring mniej doświadczonych testerów.
