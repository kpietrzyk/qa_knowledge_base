# Ścieżka: AI tools dla QA

## Poziom 1 — prompty

- analiza wymagań,
- checklisty,
- bug report review,
- test case generation.

## Poziom 2 — AI w IDE

- Copilot,
- Continue.dev,
- Cline/Roo Code,
- praca z repo.

## Poziom 3 — AI + automatyzacja

- Playwright Codegen + AI review,
- Appium locators + AI,
- Maestro flow generation,
- Allure report analysis.

## Poziom 4 — agent

- Playwright MCP,
- Appium MCP,
- Browser Use,
- ograniczone zadania eksploracyjne.

## Zasada

AI ma przyspieszać myślenie i wykonanie, ale nie może zastąpić oceny ryzyka przez testera.
