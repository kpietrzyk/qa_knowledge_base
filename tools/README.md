# Tools

Ten folder zawiera opisy narzędzi. Każdy plik ma podobną strukturę, żeby człowiek i AI-agent mogli szybko porównać narzędzia.

## Kategorie

- `mobile/` — Android, iOS, emulatory, ADB, Appium, Maestro
- `api/` — Postman, Bruno, Hoppscotch
- `web/` — Playwright i narzędzia webowe
- `ai/` — Copilot, Continue, MCP, agenci
- `debugging/` — proxy, crash reporting, raporty
- `test-management/` — Jira, Xray, Kiwi TCMS, TestLink
